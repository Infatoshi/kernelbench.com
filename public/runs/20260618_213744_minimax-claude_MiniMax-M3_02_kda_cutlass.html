KDA is a linear attention variant with **per-dimension log-space decay gates** `g`, **beta gating** on keys, and a **delta rule**-style state update. Grouped value attention (GVA) is supported via `HV = G·H`.

### Core update rule (recurrent)

Given `S ∈ R^{K×V}`, at timestep `i`:

```
S ← S · exp(g_i)            # apply per-dim decay
k̃ = β_i · k_i
ṽ = v_i − (k_iᵀ S) · k̃     # residual against current key
S ← S + k̃ ⊗ ṽ              # rank-1 outer product
o_i = q_iᵀ S
```

The novelty vs. GatedDeltaNet is that the decay `exp(g)` is **per-dimension (length K) rather than scalar**, enabling a data-controlled gating mechanism.
